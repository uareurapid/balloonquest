First Ground contains handpainted textures, materials, particles, demo scenes and sample scripts which helps you quickly create amazing Arctic Winter, Grassland, Volcano and Wasteland scenes. 

Special plus, this package comes with GUI animator FREE that lets you easily animate UI objects like a professional.

Features: 

	• 37 Colorful handpainted textures .
	• 12 Arctic textures.
	• 9 Grassland textures. 
	• 9 Volcano textures. 
	• 7 Wasteland textures. 
	• 4 Demo scenes.
	• Fog, snow, smoke, sand particles. 
	• Mobile friendly. 
	• Unity 5.0.1 or higher compatible.

Note:
	• This package can work with Unity's Terrain painting tools. 
	• You do not need this package if you have First Fantasy for Mobile 1.3 or higher (https://www.assetstore.unity3d.com/en/#!/content/10822).
